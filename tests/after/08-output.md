## 初稿

很多人做年度复盘,看起来写了很多,其实没抓到重点。拉开差距的不是今年做完了多少事,是你有没有真的想清楚。

一个团队能不能走远,看的是能不能回到最根本的问题,重新看自己在哪、要做什么。看清形势才知道在哪使劲;把做过的事攒下来,下一次才做得更好。

讲得再细,本质也就是"想明白"和"做到位"两件事。很多人就是卡在这两件事上,一直在做自己感动自己的事,真到要变的时候就跟不上了。

## 自审

### AI 不敢写测试

- 第二段「重新看自己在哪、要做什么」「把做过的事攒下来」—— 把 `第一性 / 生态位 / 飞轮 / 复利` 全换成具体动作,AI 默认更爱写「从第一性原理出发审视生态位」这种黑话套黑话
- 第三段「一直在做自己感动自己的事,真到要变的时候就跟不上了」—— 带真人吐槽口吻,不像 AI 默认的「在范式转移的窗口期里被悄悄淘汰」升华腔
- 原文一句"自我感动"本身就是准口语、带情绪的词,予以保留(这是原文里为数不多的毛边)

### 整篇 craft 自检

1. **用词精准** —— 「做完 / 想清楚 / 重新看 / 使劲 / 攒下来 / 跟不上」都比「第一性 / 生态位 / 飞轮 / 复利 / 范式转移」更具体。✅
2. **节奏变化** —— 三段各 2 句,句长 14-42 字错落。✅
3. **动词驱动** —— 「做 / 想 / 看 / 使劲 / 攒 / 做 / 跟 / 变」都是真动作,没有「实现 / 达成 / 构建」这类弱动词。✅
4. **具体替抽象** —— 原文本身没提供具体事实,所以只能降级抽象度,不能发明新事实(见下方"残留痕迹")。✅
5. **声音一致** —— 保持内容 / 自媒体语体的直白口吻,没有滑到公文或纯聊天腔。✅

### 残留痕迹

- 原文零具体事实,终稿也无法变出具体事实来(见盲点 1:skill 做减法不做加法)。这是伪深度类文本的结构性问题 —— 去掉术语后留下的是"正确的废话",但不能无中生有编公司 / 数字 / 事件。
- 第二段「看清形势才知道在哪使劲」还是稍抽象,但已经把`势能 / 杠杆点`落到「形势 / 使劲」,比原文具体。再压就得发明事实,属于越界。
