## 一、人群资产模型术语

- **5A 人群模型**:A1(了解/感知)→A2(吸引/好奇)→A3(问询/深度种草)→A4(购买/行动)→A5(复购/拥护),行业内简称"5A 人群""A3 人群"
- **O 人群**:机会人群(5A 之前的潜在触达人群)
- **正漏斗 / 反漏斗**:抖音"正漏斗"以生意为导向(O→A1→A2→A3→A4→A5 递进转化);小红书"反漏斗"以内容为导向(按平台人群标签精细刻画,由核心向外扩散)
- **人群双螺旋**:正/反漏斗跨平台人群标签迁移打法
- **AIPS**:小红书人群资产维度(认知/兴趣/购买/分享)
- **TA/TI**:目标人群(Target Audience)/ 种草人群转化(TI 维度)
- **DMP 人群包 / Lookalike**:数据管理平台圈选人群包、相似人群扩展
- **橄榄型资产结构**:相对"漏斗型"更健康的人群分布(中间层厚)
- **A3 拉新率 / A3 渗透率 / A3 流转率**:种草效率核心指标

## 三、命名方法论 / 框架(公式化命名手法,行业通用套路)

- **新 4A**:数据化策略(AI-driven strategy)/ 流量化创意(Adaptive content)/ 全域化投放(All-link channel)/ 协同化品效(Attributing effect),头条易对标传统 4A 广告公司提出的自研能力框架
- **IPO 模型**:新品上市三段式,看清(Insight 洞察)→找准(Positioning 定位)→打爆(Offerings 卖点全触点投放)
- **Q³ 法则**:内容量化标准,Quantitative(量爆,发布 3 天播放量行业前 10%)× Qualitative(质爆,发布 3 天 A3 新增率行业前 50%)× Quota(度,NPS / 看后搜率 / 评论提及率)
- **RED 法则**:小红书爆文方法论,Relevant(数据洞察)× Experience(真实使用场景)× Distinctive value(功能 + 情绪价值)
- **STP × KFS**:小红书整合营销模型,STP(市场细分-人群锁定-买点定位)× KFS(KOL/KOC/KOS 内容矩阵 + Feeds 信息流放大 + Search 搜索卡位)
- **FARM 模型**:抖音后链路转化,Field(商家自播)+ Alliance(达人直播)+ RTB(竞价广告)+ Mall(货架场)
- **星推搜直**:抖音全链路种收合一,星(达人种草)→推(信息流加热)→搜(关键词卡位)→直(直播转化)
- **1+X+N 场景营销公式**:1 个核心场景 × X 个内容维度切口 → N 个媒介触点
- **五度先验公式**:新品类赛道机会判断,心智顺应度 × 购买理由强度 × 人群画像精度 × 品类趋势浓度 → 心智阶梯饱和度
- **单品价值铁三角**:态度(品牌溢价)- 精度(市场份额)- 广度(用户规模)
- **TIPS 模型**:达人投放,Target audience(选人群)→Industry(行业校准)→Positioning(关键词占位)→Strengthen(势能 + 生意双重放大)
- **"海陆空"媒介策略**:空军(明星 / 硬广炸开心智)→海军(达人种草深度种草)→陆军(FARM 模型持续收割转化)
- **DVC**:短剧化品牌内容(Douyin × View × TVC 质感 × Commercial),品牌价值 / 情绪价值 / 传播价值三维
- **荷花模型**:达人矩阵扩散,以"荷母"(品牌 TA / 头部达人)为核心向"花瓣"资产扩散,含以人带人 / 以人带货 / 以圈层带圈层 / 以金句带金句 / 以矩阵带地域五种子模式
- **热点营销罗盘 / 冲热点与蹭热点二分法**:品牌定制专属话题(冲)vs 借势平台既有热点(蹭),对应"热点服务品牌"与"品牌借势热点"互逆关系
- **SCI 分析**:品牌认知(Scale)- 种草(Conversion)- 转化(Image)三段雷达图

## 四、平台效果指标黑话(通用,不算 AI 腔)

CPM(千次曝光成本)、CPE(单次互动成本)、CPC、CTR、CVR、CPA3(种草成本)、SOV/SOC(声量份额 / 声量集中度)、NPS(净推荐值)、TGI(人群偏好指数)、看后搜(内容曝光后主动搜索行为)、回搜率、爆文率、完播率 / 5s 完播率、质爆 / 量爆(内容测评双轴)、种草率、渗透率(品牌体量 / 大盘体量)

## 五、内容 / 创意手法命名

- **视觉锤 + 语言钉**:产品记忆点双重叠加(视觉符号 + 文字锚点)
- **小蓝词**:评论区 / 视频底部可点击搜索关键词组件,用于承接"看后搜"
- **内容黄金公式**:强冲突场景 + 生活化建信任 + 痛点触发 + 核心卖点 + 情感化收尾(A3 增长型);强共鸣场景 + 潜在问题暴露 + 解决方案 + 优势展示 + 理念升华(看后搜型)
- **人格化文案体**:以品牌名谐音结合社交关系词自造专属话题体裁,形成品牌独有传播语体
- **"用 X 品 X 抖 X 梗"**:头条易内部高赞商业评论公式
- **五榜联动**:挑战榜 / 综艺 IP 榜 / 区域榜 / 种草榜 / 定制热点榜

## 七、文档结构 / 修辞手法(可复用于提案与方案写作)

- **公式化命名法**:把营销打法压缩成朗朗上口的简称 / 公式(字母缩写、数字口诀),配 3-5 个关键维度括号定义句,便于口播记忆与客户复述
- **案例卡片黄金结构**:项目背景 → 项目亮点 → 数据成果三段式,或"背景-亮点-数据"三段论
- **"痛点错位"起手式**:多个案例背景用"XX 与 XX 的错位 / 断层"起笔,再引出"需要重新 XX"的解决方案钩子
- **军事隐喻分层法**:用"空军 / 海军 / 陆军"类比传播链路不同层级,把抽象投放链路具象化
- **提问-解答体(Q&A)结构**:以"Q:行业痛点是什么 / A:我们的方法论"开场,制造"精准命中问题"的说服节奏
- **量化模型的三元乘法结构**:多次出现"维度 A × 维度 B × 维度 C → 综合判断"的公式化表达,把定性判断包装成可计算公式,增强说服力
- **"总-分-总"引言页打法**:开篇两句话立住核心论点,左右栏拆解短期 / 长期怎么做,末句收束成一句可传播的 slogan
- **"痛点→解法"强对比页面**:先用极简设计砸出行业困境,再给出"我们开发独家模式"的解法页,形成问题-方案叙事张力
