## 反复出现的行业黑话与方法论命名

- **定位话术**:「中国短视频 4A 全案营销开创者」「双品效全案运营商」「TOP 级内容营销服务商」
- **4A 四要素英文命名法**(值得复用的定位公式):AI-driven strategy(数据化策略)/ Adaptive content(流量化创意)/ All-link channel(全域化投放)/ Attributing effect(协同化归因),即"策略—创意—投放—归因"四段论,每段配一句核心主张
- **品牌赋能模型**:「双引擎增长」= 品牌增长(传播力/拉新力/种草力/搜索力)+ 生意增长(产品力/渠道力/渗透力/复购力),左右两翼对称结构,是一种常见的"品效合一"论证框架
- **数据资产术语**:数据银行、5A 人群资产、A3 人群(拉新人群分层)、人群拉新率、种草率、CTR(点击率)、种收一体、品销协同、全链路科学营销策略
- **数据工具方法论命名**:「数据管理—数据洞察—数据交付—数据归因」四段闭环;TIPS 方法论(T 达人精选、I 检索占位、P/S 转化赋能,原文字母对应有缺失但结构是"达人筛选→搜索占位→转化验证"三段式)
- **案例分类话术**(用于案例集锦分组小标题,句式统一为「我们懂 X:一句话价值主张」):我们懂字节 / 我们懂数据 / 我们懂内容 / 我们懂玩法 / 我们懂电商,每组配 2 个对比案例
- **事件营销命名套路**:案例标题惯用「品牌名+现象级+品类关键词+大事件」句式(如「现象级品牌传播大事件」「现象级新品传播大事件」),强调"现象级"作为效果背书词

## 可复用的结构与框架

1. **公司介绍标准页序**:一句话定位 → 业务范围/组织规模 → 行业资质背书(平台官方认证列表)→ 服务客户 logo 墙(用规模感建立信任)→ 核心方法论模型图 → 独家工具介绍 → 汇总性数字全景图(供公开的规模性数字,不涉及单项目机密)→ 案例集锦 → 荣誉墙 → 创始团队背书 → 致谢页。这是一套通用的乙方公司通案骨架,可迁移到其他 To B 服务型公司的自我介绍文档。
2. **单案例卖点结构(BACKGROUND-IDEA/STRATEGY-RESULTS 三段式)**:背景(客户面临什么节点/痛点)→ 打法(用什么钩子/IP/组合拳,常配一句"以 XX 为抓手"的动词句)→ 结果(先讲平台维度荣誉如上榜时长,再讲汇总性传播数据)。这是通用的案例页论证骨架,背景段负责建立张力,打法段负责给出"钩子词"(一个可传播的话题标签或悬念点),结果段负责多维度证据交叉验证(不止一个指标)。
3. **案例证据分层手法**:案例效果展示习惯把"量"(播放/曝光规模)和"质"(上榜次数、行业排名、官方奖项)分开陈列,即"量爆+质爆"双轨验证,避免单一数字支撑说服力。
4. **对比式并置排版**:同一页左右两个品类不同的案例并置对比(如医药类 vs 快消类、美妆类 vs 汽车类),强化"跨行业通吃"的能力主张,是一种常见的乙方通案排版手法。
5. **团队背书写法**:创始团队页固定给每人配「曾任职机构 + 从业年限 + 专长领域」三要素式简介,技术/内容/运营/增长四类高管分工清晰,体现团队专业互补性的叙事手法。

# 来源：头条易《2023 年汽车行业营销通案》（章节：2、行业通案）｜kind: template

## 反复出现的行业术语 / 平台指标名 / 方法论命名

- **能力/资质类**：营销科学金牌服务商、全效服务商、DP 品牌服务商、星图最佳合作服务商、品效销全链路
- **平台工具/产品名**：巨量云图、巨量星图、巨量千川、数据银行 5A 人群资产、MarKol（头条易自研达人营销工具）、种草通、车云店、轻任务、双端直播、优选线索、搜索品专
- **方法论命名**：双话题引爆法、TIPS 达人投放模型（Target audience / Industry / Positioning / Strenthen）、A1-A5 人群资产模型（感知-吸引-询问-行动-拥护）、内容工厂（三步：批量筛选优质达人→批量生产优质内容→沉淀品牌达人库）
- **卖点提炼公式**：产品卖点 + 使用场景 + 亲身感受 = 内容共鸣；情绪共鸣（3S 切入梗：好奇/恐惧/贪婪/有用/从众/攀比/叛逆）+ 人群痛点 = 内容模型
- **场景术语**：口碑营销-内容场、互动营销-社交场、用户转化-转化场；种草到割草；私域资产沉淀；A3 流转率

## 可复用的结构 / 框架

1. **三场闭环框架（内容场-社交场-转化场，用户居中）**：以用户为核心画一个漏斗+环状结构，内容场负责「与用户建立情感链接」，社交场负责「构建长期关系」，转化场负责「促使用户产生行动」，三场外圈用达人营销/双话题/搜索品专/口碑舆情（内容场）、轻任务/车云店（社交场）、双端直播/优选线索（转化场）分别落地，底部用巨量云图统一承接数据。这是一个可复用的「获客-养熟-转化」全链路论证骨架，不限于汽车行业。
2. **四大优势能力页面结构**：讲公司实力时用「数据驱动能力+专业服务团队+达人资源+全链路经验」四宫格，每项配一句概括+数据背书，是标准的公司能力介绍页套路。
3. **单案例卡片结构**：图标/客户 logo → 一句话标题（品牌 X 打法名）→ 项目背景（一段话说清品牌困境或目标）→ 执行亮点（2-3 条编号列表）→ 效果数字（2-4 个大号数字卡）。这是甲方案例展示的通用排版模板，可复用于任何行业案例页。
4. **内容模型四层拆解表**：场景（通勤/出游/玩乐/接待等）× 场景细分 × 车型卖点 × 产品支撑 × 用户利益点，做成矩阵表格，是「产品卖点场景化」的通用方法，本质是把抽象卖点翻译成具体使用情境下的用户语言，可迁移到其他耐用消费品类。
5. **达人分层投放漏斗**：TIPS 模型体现的是「先圈人群（TA）→再按行业校准（Industry）→再抢占关键词心智（Positioning）→最后双重放大势能与生意（Strenthen）」的递进逻辑，是达人投放从粗到细的通用决策链。

## 可复用的结构/框架(方法论,供未来项目复用)

1. **公司能力介绍页的固定套路**:一句话定位(XX领域开创者)→四大/N大核心优势并列卡片→逐条展开(每条配一页详解,标题采用"编号+能力名+价值主张"三段式,如"01 投数合一科学营销——头条易坚持投数合一,赋能品牌营销更高效")→荣誉奖项佐证(奖项名+年度,不列具体金额)→团队介绍(城市覆盖+从业年限+能力标签,不含真实姓名)。
2. **行业背景铺垫的三段式时间轴**:营销发展阶段划分(线下拓客期→传统媒体投放期→互联网精细化运营期),每段配"渠道特征+效率痛点+转折点",用于论证"为什么现在该做XX打法"。
3. **人群运营漏斗骨架**:知道品牌→熟悉想要产品→相信产品→爱上产品,四阶段分别对应"用户感受-内容形式-达人/流量策略-人群复用"四行矩阵,是一个可复用的品效销通用表格框架,不限于家居行业。
4. **人群偏好画像表的列设计**:人群分层(如按年龄/身份分组)× 偏好内容特点 × 核心消费品类 × 细分品类TOP3 × 偏好内容关键词,五列结构可直接套用到其他行业的人群策略页。
5. **案例页的固定叙事公式**:客户所属品类(脱敏后表述)+一句话结果导向标题→"TIPS/达人精选-搜索占位-转化赋能"式三段能力标签→亮点复盘(定性表述为主)。这是案例页最值得复用的"三段论"结构:打法归因(用了什么方法)→定性效果描述(增长趋势/量级变化用模糊区间而非精确数字)→复盘一句话总结。
6. **调优策略三步法**(用于线索留资/投放优化类项目):定向&内容测试期(AB test筛选优质素材及定向)→优质定向&内容放量期(取优放量)→平稳放量投放期(找共性避免掉量、加大放量)。配套"受众定位→场景设计→产品导流"三角图。
7. **矩阵搭建的阶段化描述**:初级阶段(总部直链经销商)→中级阶段(标杆带区域)→高级阶段(专业化矩阵),用金字塔图呈现"1个品牌总号+N个企业自号+X个经销商直营号"的分层扩张逻辑,可复用于其他"总部-区域-末端"型渠道打法的可视化。

## 案例部分的方法论提炼(kind涉及case片段,账户/平台展示不算脱敏对象)

- **美妆案例**:方法论亮点是"By TA洞察画像锁定人群+定向投放"与"结合热门关键词/场景卖点/痛点及产品关键词加挂小蓝词"两者组合,策略角度的可取之处在于把达人投放的人群定向与搜索占位词库做了强绑定,而不是分开执行;效果验证角度用的是"投前/投后曲线对比"呈现方式(倍数增长趋势图),而非单一时点数字,这种「投前投后对比图」的呈现手法本身可复用。
- **汽车案例**:方法论亮点是"星图+搜索"链路的首次打通,策略核心是"品专承接、跑通完整星→搜链路",即达人内容里嵌入的搜索词最终要有承接页(品牌专区)接住,形成种草到转化的闭环设计思路,这是一个可复用的"内容-搜索-承接"三级衔接检查清单。
- **家电案例(核心场景精细化式)**:方法论亮点是"核心场景精细化"打法——先按消费决策路径圈定核心垂类达人(如某品类围绕"母婴/家居/评测"三类关联受众),再围绕单一核心话题从不同角度制造共鸣内容,创意角度的可取之处是"存量竞争市场里,用KOL科普+新品深度测评+垂类达人内容三层组合,从0到1抢占一个新品类心智",这种"聚焦单点新品类打透"的创意策略适合存量红海行业参考。
- **家电案例(挑战赛整合式)**:方法论亮点是"挑战赛+全民任务+定制贴纸+KOL分发+投后云图数据赋能"的组合打法,策略角度可取之处是"话题设计+互动设计+达人策略+投放策略=组合出击、效果叠加"这一原则本身,即单一维度打法效果有限,必须多杠杆同时启动才能产生"长尾效应"。案例还提供了"探店类达人"的价值主张模板:到店体验+产品卖点介绍+活动信息传递三合一,适合兴趣人群的声量传播场景(而非转化场景)。

**IPCA 种草四步法**：Insight 科学洞察（赛道 / 竞品 / 本品三层洞察）→ Plan 正确规划（指标预设 + 预算匹配）→ Carry 内容共创（达人策略 + 内容方法论）→ Amplify 流量放大（内容放大 + 卡位拦截）。是一个可迁移到任何内容平台的「洞察-规划-共创-放大」通用打法骨架。

## 核心框架:3i 法则

内容围绕小红书"双列信息流"(一屏 4-6 篇笔记、封面+标题组合触达用户)这一平台机制展开,提出**3i 法则**作为总纲:
- **Visual impact(视觉冲击)**→封面
- **Reinforcement impression(强化认知)**→标题
- **Valuable information(价值驱动)**→内容

三者关系被明确表述为"紧密结合、相互补充的整体关系,需相辅相成采纳,而非单一元素堆砌产出"。内容价值进一步拆为四种:实用价值、情绪价值、信息价值、决策价值。这是一套可直接复用于"内容三段式诊断/创作检查清单"的通用骨架,不限于小红书,略作变体即可套用到任何"信息流+标题党"型平台的内容生产。

## 1i-封面:类型学与选择框架

**五种封面类型**:人物型、产品型、场景类、图文结合型、特殊类型。

**四项硬性要求**(检查清单式):①突出创作主题(反例:客户要 high energy 啤酒卖点,达人却穿优雅长裙在清吧拍摄——文不对题);②色调贴合主题(强调"饱和度≠明度",高饱和不等于画面亮,美食类"调高饱和度增强食欲感"是几乎通用的执行技巧);③主题与产品卖点关联(软植允许不明显露出产品但需强关联,硬植/堆叠放大"考验拍摄手法,硬植有风险");④画面信息主次分明(花字颜色用邻近色或互补色突出主次)。

**五项风格选择**(持续更新型清单,说明这是内部活文档):夸张高饱和撞色、渲染氛围感(细分出"精致名媛贵妇都市丽人""复古 y2k""情侣双人性感"三个子风格,并提示"突出情欲适用于情侣或性感氛围,但平台审核严格需把控尺度")、强对比反差(适用美妆家装的前后对比)、特殊封面(萌宠类、独特新奇标新立异类,提示"不建议没有创作能力的作者盲目跟从")、透视夸张大广角("发疯文学"拍摄梗,提示"达人需要一定的幽默感和拍摄技术")。"氛围感"被给出一个极简操作定义:"光感强烈(发光)的照片=氛围感"。

## 2i-标题:九种标题类型学

利益型(横评/教程,理性诉求+数字)、感叹式(避免极限词,用"关键""重点"替代)、疑问式、热点型(站内外热点结合节日节点)、互动型(设问/吐槽勾起倾诉欲)、引战型("对固有认知提出挑战""适合走黑红路线")、共鸣型(回忆杀/地域标签/外国人视角打文化差异)、故事型(依赖画面和达人审美水平)、悬念型。附有一份可直接复用的"爆款标题关键词库"(如:好用到哭、大数据、教科书、小白必看、YYDS、我不允许、压箱底、谁懂啊、邻居看了要抄作业、妈妈说这是智商税等)。

## 3i-内容:三要素通用模板 + 排版规范

**内容基本要素**:核心产品点(卖点应出现在内容三分之二篇幅、产品名首次提及需完整清晰)、人群定位及情绪(需体现真实分享情绪)、场景表述(即使无法画面呈现也要用语言表达)。给出错误/正确表述对照示例,方法论是"把产品功效转译成第一人称生活场景叙事"而非直接罗列功能。

**排版三原则**:口语化表达引共情、emoji 强化结构信息、格式统一主次分明。提及官方排版工具"红薯编辑器"(微信小程序)及"PK/投票互动组件"用于提升争议型内容互动量。

**关键词布局**:核心产品词(必现)+ 类目词 + 场景词 + 功效词 + 人群词,分主次两级布局思路,用于搜索引流(文档强调"小红书日均搜索量已达 3 亿次,超过 60% 用户的首个动作是搜索")。

## 高阶创作手法(五种可复用套路)

1. **热点融梗**:区分"站内热点"(活动通知、搜索热门榜)与"站外热点搬运"(利用小红书信息滞后性,把知乎/豆瓣/微博/抖音已发酵话题搬运过来打时间差,或搬运海外流行文化);强调"融入产品特色或切换视角、加入自身创意点"而非硬蹭,并给出一个反面警示案例的分析路径:识别一个话题因"重复率过高引发审美疲劳"而产生的舆论反转(争议话题词从"人鱼公主"演变为带贬义的"流水线公主"),点出其"带性别歧视色彩、争议性强"的风险属性,以此说明蹭热点前需先判断话题的情绪极性和风险,而非只看热度。
2. **发现问题→解决问题**:制造/发现焦虑或痛点,再用植入产品收口。经典结构是"开篇罗列生活小问题→加入第二视角强化痛点(如家人的抱怨)→植入产品完美解决"。
3. **两种人群标签的差异或矛盾**:引入第三人称视角讲产品,或制造对立观点再"打脸",标签维度包括代际关系、地域差异、阶层差异,但落点必须是"通过展示产品卖点达成意见统一"——即制造冲突只是钩子,收口要和解。
4. **贴标签**:给人群或行为贴标签(借用已有热词如十二星座、MBTI,或自创概念),用于精准圈定受众、引发情绪共鸣,但明确提示"需拿捏尺度否则会被平台限流",并列出反面标签清单(普信男、性别对立、色情擦边)作为红线参考。
5. **反转式结尾**:视频笔记中加入意想不到的结局提升完播率(考验幽默感和创意能力)。

## 方法论价值小结

这份材料的核心方法论价值在于:把"小红书笔记为什么会火"这个模糊问题,拆解成了封面/标题/内容三个可独立打磨、又强调必须联动的模块,每个模块都配"类型学清单+硬性要求+反例警示",是一套典型的"代理公司内部培训型方法论文档"结构——不针对单一客户,而是沉淀成团队通用的创作检查清单和黑话词典(3i 法则、图文结合型封面、氛围感、软植/硬植、黑红路线、发疯文学、蓝海词等)。这套"类型学枚举+反例卡点+关键词库"的文档组织方式本身,也是可迁移到其他平台方法论沉淀的通用结构。
