# 标点符号 · 第二层规范：每个标点干什么

> 本文件是 qu-ai-wei 的 **punctuation deep reference**。SKILL.md 的「标点符号:中文正文一律用全角」一节解决了"用对字符"的问题(全角 vs 半角),本文件解决"每个标点干什么"。
>
> 以 GB/T 15834-2011 的标点功能为规范依据,结合文体与上下文判断。本文不提供未经语料统计验证的「真人频率基线」,标点本身也不用于判断作者身份。

---

## 顿号「、」vs 逗号「,」：按并列层级选择

- **顿号：** 并列的词、并列的短语、并列的名词。`苹果、梨、香蕉` / `降低成本、提升效率、改善体验`
- **逗号：** 句子内部的停顿、分句连接。`我去买了水果,然后回家。`

**常见功能错配：** 简单并列词之间全用逗号(`苹果,梨,香蕉`),没有按中文并列层级选择顿号。

并列词语和短语通常用顿号;并列分句、内部已有顿号的复杂项或需要更强停顿时,按国标改用逗号 / 分号。不能用「并列一律顿号」覆盖所有结构。

---

## 分号「;」：排比句群的节奏工具,语体敏感

分号跟破折号不同,高密度使用也不一定有问题,需要分层看。

> ⚠️ **语体分层触发：**
>
> - **日常 / 自媒体 / 社交 / 商务 / 邮件 / PRD**:高密度分号若没有分层功能,改用句号或逗号;有清楚并列层级时保留。
> - **社论 / 时评 / 长议论 / 学术论证**：分号用于**结构对称的排比句群分层**,是合法修辞。《南方周末》新年献词经典结构是连用多个分号分隔排比段落（`...;...;...。`）,这是严肃论述体的标准节奏,**不改**。
> - **判据：** 分号两侧是不是**结构对称的句子**?对称 → 合法排比,保留;不对称 → 英文句法硬拼,改。

---

## 书名号「《》」：覆盖面比一般人以为的更广

严肃媒体的书名号覆盖范围远超"书"本身:

- **刊物**：《三联生活周刊》《财新周刊》《爱乐》《读书》
- **书籍、电影、音乐作品、MV**
- **政策文件、法规、政府报告**：《中共中央关于制定国民经济和社会发展第十五个五年规划的建议》
- **栏目名、专题名**：《拆新闻》《从现场出发的传播学》
- **年度产品 / 特定作品**：《2025 三联生活历》

**常见功能错配：** 用引号 `""` 代替书名号(`"人民日报"`),或把应标明的作品名直接裸写。判断依据是对象类型与目标载体规范,该项属于格式风险,不用于推断文本来源。

## 省略号「……」：按省略 / 未尽功能判断

省略号合法出现的场景:

- **对话 / 口述实录**里的未尽意：`"我也不知道……就这样吧。"`
- **列举未完的"等等"义**：`春天的香椿、夏天的荷花、秋天的月……`
- **文艺 / 散文 / 营销文案的节奏需要**
- **小红书 / 朋友圈 / 微信私聊**等随意语体

严肃议论 / 新闻 / 财经正文里若省略号不承担省略、列举未尽或引语停顿功能,再按文体改写;单次合法使用不触发。

**格式规范：** `……`（两个 U+2026,占两字宽）,不是 `...`（三个半角点）、不是 `。。。`、不是 `. . .`。

---

## 冒号「:」vs 破折号「——」：引出信息是冒号的活,不是破折号

严肃媒体里,凡是"引出"动作 —— 引语、列表、署名、副标题、小标题 —— 默认冒号,不是破折号:

- **引语**：`他说:"我不知道。"`
- **署名**：`责任编辑:XXX` / `发自:重庆`
- **小标题**：《财新周刊》`秦泗钊:要学机器不会做的事`
- **列表起首**：`这次改版重点:登录流程、bug 修复、首页布局`

破折号承担的是**补注、强调、转折、节奏**,不是引出。

**常见功能错配：** 把引出关系 `X:Y` 写成 `X——Y`,尤其是「他说——」「报告指出——」。只按标点是否承担引出 / 补注功能判断,并与规则 #16 / #25 互锁。

---

## 引号「""」的三种合法用途

引号的三种合法场景:

1. **直接引语**：`她说:"明天再来。"`
2. **概念 / 术语的标引**（尤其政治、经济、专题术语）：《财新周刊》用 `"十五五"` `"钱袋子"` `"同案不同判"` 标引简称、比喻性表达、专题名
3. **反讽 / 特殊用法**：`这就是所谓的"优化"`

**常见功能错配：**

- 在每个普通名词上加引号（`"用户"打开"应用"使用"功能"`）
- 用双引号代替书名号（`读"三联生活周刊"`）
- 在非引语场景给动作 / 状态加引号（`他"思考"了一下`）

**嵌套规则：** 外层双引号 `""`,内层单引号 `''`。`他说:"她告诉我'别走'之后就挂了电话。"`

---

## 括号「()」：次要补注,不引出主干

括号内容是**可删除不影响主干**的次要信息。严肃媒体主要用在:

- **英文缩写补注**：`世界超级摩托车锦标赛(WSBK)`
- **时间 / 地点 / 来源**：`(2026-01-29 发稿,北京)`
- **数据来源、署名**：`(作者名 / 视觉中国)`
- **次要解释**：`这个工具(早期叫 Bard)很好用`

**跟破折号的分工：** 括号 = **次要补注**(读者可跳过),破折号 = **重要补注 / 强调 / 转折**(读者不能跳过)。如果全部改用破折号,次要和重要信息会被压成同一强度。

---

## 标点检查顺序

1. 字符是否符合目标载体与用户规范。
2. 标点是否承担国标规定的功能。
3. 同一标点是否密集重复同一种无效结构。
4. 文体是否有意使用该节奏,如特稿补注、品牌短句、口述停顿。

只有第 2 / 3 项出现明确问题时才进入表达诊断;第 1 项只是格式风险,第 4 项是保护条件。
