# 品牌广告 / 文案语体（深度参考）

> 本文件是 qu-ai-wei 的 **brand voice deep reference**。SKILL.md 的第零步在识别到「品牌广告 / 文案」语体时,读取本文件以获取品牌文案的语体特征、跟自媒体的区分线、特殊判定与工作示例。
>
> **非品牌语体不需要读此文件。**

---

## 为什么品牌广告 / 文案要单独列一种语体

品牌官网 hero copy、campaign 主视觉、发布会大字与户外文案,跟这个 skill 里其他 8 种语体有明显差异:

- **不是自媒体营销号** —— 没有 KOL 人设、没有"姐妹们冲"、没有点赞收藏 CTA、没有伪深度咨询腔；品牌是企业级单向输出,不是"我"在跟你聊。
- **不是商务 / 职场** —— 品牌文案避免"赋能 / 助力 / 打造 / 闭环"这类商务僵尸词,它们恰好是品牌文案要**反着写**的东西。
- **不是书面 / 一般** —— 品牌文案可用极短句、不完整句、留白、文言感与中英混排;这些放进博客时评里未必合适。

### 品牌文案的语体特征

1. **高度压缩**:一句只承载一个动作或感受,如「轻一点。快一点。」
2. **祈使动词 + 物理动作**:`上`、`冲`、`跑`、`跳`、`敢` —— 不是`激发 / 赋能 / 助力`
3. **允许排比、韵律、文言感** —— 这些在书面语里可能显得机械,在品牌语境里可能是刻意设计的修辞
4. **允许留白与不完整句** —— `轻。薄。快。`、`没状态?上。没把握?上。` —— 这些在#35 句长均质化里是"短句过短",但品牌语体里是节奏
5. **中英可混用** —— 产品名、技术名按品牌官方写法保留,普通英文词仍看受众
6. **文案里没有"我"(KOL 人设),只有"你"(品牌对你说)或无主语祈使**

**所以品牌文案要单列一档**,否则按其他 8 种语体任何一种都会误改：当商务会被加回"赋能助力";当自媒体会被塞回"家人们";当书面会被改成完整长句,破坏节奏；当特稿会被扩展成段落,杀掉留白。**判断不清时,问用户:"这是要发在品牌官网 / 广告片 / 发布会大字 / 户外大牌上吗?"** 是 → 切品牌广告语体。

## 品牌广告 vs 自媒体营销号：三条识别线

这两个语体最容易混——都偏消费端、都"卖东西"。但它们是不同物种,激活的规则子集几乎反向。区分看三条：

| 信号 | 品牌广告 / 文案 | 自媒体营销号 (#37) |
|---|---|---|
| **说话的人** | 企业级单向输出,通常没有 KOL 式「我」 | 有强人设,使用第一人称 KOL 视角 |
| **CTA 形式** | 短祈使 + 行动动词 | 点赞、收藏、关注、评论等流量导向 |
| **信息密度** | 高度压缩,留白服务视觉与节奏 | 填充式(痛点 → 方法 → 升华),文字堆得很满 |

**四个边界场景：**

- 品牌**自己运营的公众号推送**,如果带 KOL 语气("今天想跟大家聊聊...")→ 走 **自媒体语体**,不是品牌广告
- 品牌**PR 新闻稿 / 年报 / 公关声明** → 走 **商务 / 书面语体**,不是品牌广告
- 品牌 **SCRM 社群「亲~」话术** → 走 #37-E 的客服亲亲体,不是品牌广告
- 品牌 **官网 hero copy / 发布会大字 / 广告片旁白 / 户外大牌** → **品牌广告语体**

## 品牌广告语体下每条规则的特殊判定

品牌广告 / 文案的雷区最多，几乎每一类都要重新判（这正是它必须单独列一档的原因）：

- **A 内容拔高**:`#1` 过度拔高意义和 `#4` 尾巴式空洞总结要删;`#5` 模糊权威归因仍要核验来源,不因品牌身份豁免事实风险。
- **B 语言**:「赋能 / 助力 / 打造 / 闭环 / 护航」只有空转时才删。产品名、技术名按官方写法保留;普通中英括注仍由受众决定。
- **C 修辞**:`#21` 四字词堆叠、`#22` 排比段要克制判断。紧扣情绪和行动的韵律保留,脱离内容的空转排比才处理。`#24` 加粗与 emoji 按目标载体判断;`#25` 破折号看实际功能。
- **E 填充**:`#32「让我们一起……」`要区分具体 CTA 和空号召。判据是能否指向明确行动,不是句子长短。
- **G 篇章**:`#35 句长节奏均质化` **反向触发** —— 品牌文案要的恰恰是节奏变化、允许超短句和留白；`#36 指代不敢省` 在品牌文案里**不触发**（品牌名有时必须反复出现做记忆强化,`iPhone × 3` 是常规操作）。
- **H 平台场景**:`#37` 自媒体套路不直接套用;`#38` 的空意象只有在脱离产品、场景和行动时才触发。

**一句话总结品牌文案规则**:允许并保护短句、留白、排比、文言感和必要的中英混排;仍要核对事实、受众与信息是否具体。

## 品牌广告 / 文案语体锚点

本项目不把某个品牌的未核验 campaign 语句当规则证据。需要品牌锚点时,应由用户提供已发布文案或从品牌官方页面取样,再按事实保护区锁定。

可用的本地语体锚点:

- 极短句:`轻一点。快一点。`
- 行动动词:`跑。跳。去做。`
- 中英混排:`为 Product X 打造`
- 留白:`这一次,只改一个地方。`

### 怎么用

1. **反向诊断**:改写后的句子是否符合目标品牌已有声口?没有用户样本时不要擅自模仿具体品牌。
2. **检查信息与形式**:短句是否承载信息,产品名是否遵守官方写法,标点与 emoji 是否适合目标载体。
3. **不做正向仿写**:品牌战略、用户洞察与制作语境不在原文时,只做文字清洁,不补造 campaign 概念。

## 5 组工作示例

下面 5 组「原文 / 改后」对照演示**品牌广告语体**的改写方向。**重点：不要把这个语体里的"改后"当成默认改写模板往其他语体上套**——它们只在品牌官网 hero copy / campaign 主视觉 / 发布会大字 / 户外大牌 / TVC 旁白这些场景才成立。

> ⚠ **输出格式说明:** 本节 5 组示例演示**品牌语体的语言 register**(短句 / 留白 / 文言感 / 中英混排),不演示完整输出流程(初稿 → 自审 → 终稿 → 打磨报告 → 改动摘要)。这里为了聚焦语体差异,只保留改动摘要;实际使用时,品牌语体的终稿同样要附「打磨报告」,格式见 SKILL.md「主动打磨 · 打磨报告」章节。

### 示例 1：产品发布 hero copy

**原文（空泛品牌文案）：**
> 全新 XX 手机,搭载了最新一代的先进处理器,为您带来前所未有的极致性能体验。无论是工作还是娱乐,都能让您得心应手,尽在掌握。重新定义您对移动设备的想象,开启智慧生活新篇章。

**改后（品牌广告语体）：**
> 新一代芯片。
> 日常应用,更快。

**改动摘要（品牌文案视角）：**
- 删 `搭载了最新一代的先进处理器` → 直接 `新一代芯片`（品牌级文案不用"搭载"这种 TO B 公关腔）
- 删 `前所未有的极致性能体验 / 得心应手 / 尽在掌握 / 重新定义 / 开启智慧生活新篇章` 等空泛总结
- 删 `您`,改为无主语短句;若品牌规范要求「您」则保留
- 长句拆短,留白做节奏
- 保留原文已有的「更快」,不补写省电或其它性能

### 示例 2：运动 campaign 主视觉文案

**原文（空泛运动文案）：**
> 我们坚信,通过不懈的努力和拼搏,每一位运动员都将实现自我超越,谱写属于自己的辉煌篇章。让我们一起,见证这场关于梦想与荣耀的伟大征程!

**改后（品牌广告语体）：**
> 去练。
> 去拼。
> 再超过昨天的自己。

**改动摘要：**
- 删 `我们坚信 / 通过不懈的努力 / 谱写属于自己的辉煌篇章 / 让我们一起 / 见证伟大征程` 等空泛铺陈
- 改成三个动作短句,仍围绕原文「努力 / 拼搏 / 自我超越」,没有补造比赛场景
- 短句连排在这里承载动作,不按 #35 的连续短句造戏剧感处理

### 示例 3：节日 campaign

**原文（空泛节日文案）：**
> 在这个充满爱的特别日子里,让我们一起回忆妈妈为我们付出的点点滴滴,感恩她给予我们的无私的爱与温暖。愿每一位妈妈都能收获属于她的幸福与美好!

**改后（品牌广告语体）：**
> 她是妈妈。
> 今天,想起她做过的那些事。

**改动摘要：**
- 删 `在这个充满爱的特别日子里 / 让我们一起回忆 / 感恩 / 无私的爱与温暖 / 愿每一位妈妈都能收获属于她的幸福与美好` 等空泛节日模板
- 把空抒情压回原文已有的「妈妈付出」,没有产品信息时不补造镜头或手机场景

### 示例 4：户外大牌短 slogan

**原文（空泛 slogan）：**
> 卓越品质,智慧生活,助力您开启未来科技之旅!

**改后（品牌广告语体）：**
> 为更好而生。

或

> 你要的未来。已经在你手里。

**改动摘要：**
- 删空泛商务词(`卓越品质 / 智慧生活 / 助力 / 开启未来科技之旅`)
- 压缩到户外载体能快速读完的长度,不设固定字数
- 允许不完整句（`你要的未来。已经在你手里。`）—— 句号做节奏停顿

### 示例 5：产品介绍（处理中英混排）

**原文（机械中译的产品介绍）：**
> 搭载变压器模型架构的智能芯片,配合全新空间计算视窗,让您体验前所未有的智慧交互。

**改后（品牌广告语体）：**
> Transformer 架构。全新空间计算。
> 交互,重新想象。

**改动摘要：**
- `变压器模型` → `Transformer`,遵守行业通用名;其他产品名仍以品牌官方写法为准
- `搭载 / 配合 / 智慧交互 / 让您体验前所未有的` 中没有信息的部分删掉
- 拆成短句 + 句号收尾

> ⚠️ **这 5 个示例的「改后」版本只在品牌广告语体里适用。** 把这种极压缩短句风格套到博客、特稿、学术论文、工作邮件上,会丢失必要信息。语体识别先行,改写风格后选。
