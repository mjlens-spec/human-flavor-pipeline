## H. 平台场景与文体

### 37. 自媒体 / 社交平台文案套路

公众号、小红书、短视频口播、朋友圈各有高频模板。本节只判断这些模板是否造成空转、语体错配或信息不足,不据此判断作者身份。

> 老式惊呼模板和伪疗愈 / 伪深度模板可能在同一段叠加。判断时只用密度、结构和信息损失触发,不按年份或单个热词定罪。

**A. 公众号：**

**老套路（已减少，但偶现）：**
- `震惊!99% 的人都不知道的...`
- `原来 X 竟然可以这样`
- `紧急提醒:再不 X 就晚了`

**新 AI 腔：伪深度咨询腔**（2024-2025 主流）
- **咨询黑话堆砌：** 逻辑闭环、颗粒度、心智、叙事、底层（逻辑）、生态位、长期主义、信息茧房、认知突围、复利、飞轮、杠杆、势能、第一性原理、范式转移
- **结构固定：** 抛出普遍焦虑（职场 / 婚恋 / 搞钱）→ 引入 2-3 个术语（不解释原意，直接挪用）→ `不是 X，而是 Y` / `拉开差距的其实是` / `底层逻辑是` 做伪洞察 → 结尾升华到"长期主义 / 自我迭代 / 认知升级"
- **诊断关键：** 抽掉术语后**没有任何具体事实、推理或经验**。问题是信息空转,不是术语本身。
- **判断法：** 随手挑一段，替换术语成同义空泛词（"逻辑闭环" → "想明白"），看看有没有掉信息。没掉 —— 就是伪深度。

**B. 小红书：**

**老套路（2025-2026 仍大量复用,不是"绝迹"）：**
- `姐妹们快冲!这个真的绝了💖` / `不好吃回来打我` / `闭眼入就对了`
- `家人们谁懂啊!` + 密集 emoji `✨🎀💖🌸👍🔥`
- 刻意惊呼:`救命! / 慌了! / 天哪!` / 开头 `🆘` emoji 替代文字版 `救命`
- **夸张量化吹牛**:`显瘦 10 斤不是梦 / 准时下班不是梦 / 一周重回好皮肤 / 人均 50r 吃到撑`
- **万能赞美词堆叠**:`宝藏 / 神器 / 天花板 / 绝了 / 绝绝子 / 绝绝 / 好吃到想原地开店 / 气场两米八 / 小心机`—— 短段 <300 字 ≥3 处即触发,脱离具体内容、换任何话题都能套
- **身份标签 + 反向 CTA 组合**:`姐妹们快冲` / `不好吃回来打我` / `打工人必看` / `租房党必看` / `学生党必看` / `敏感肌姐妹` / `宝妈必看` —— 身份标签 + `快冲 / 闭眼入 / 必看 / 必入 / 码住` 组合高频,密度 ≥2 组即触发
- **info-block emoji 微模板**(槽位化四字小标题 + emoji 前缀):
  - 探店:`✅ 必点招牌 / 📍 地址 / ⏰ 营业时间 / 💰 人均`
  - 好物:`✅ 好物清单 / 💰 价格 / 📦 使用场景`
  - 护肤:`🌟 晨间护肤 / 🌙 晚间护肤 / 💡 Tips / ❗ 避雷`
  - 教程:`✅ 技巧 1 / ✅ 技巧 2 / ✅ 技巧 3`
  - 跟 #18 过度数字化列点(1️⃣2️⃣3️⃣)分工:#18 是**编号**模板,本条是**命名 + emoji 前缀**模板。两条叠加即高警觉。

**区分自然俚语(保留)与机械堆叠(改):**

- 保留:`闭眼入 / ins 风 / 码住 / 种草 / 精致 / 小心机`(单独出现且语义准确时保留)
- 改:`姐妹们快冲!不好吃回来打我!`(连续号召没有增加信息)
- 改:`租房党必看 + 闭眼入 + 幸福感爆棚`(身份标签 + 俚语机械叠加)
- 改:`宝藏 × N / 神器 × N / 天花板 × N`(密度过高,万能赞美词脱离内容)

**伪疗愈 / 伪搞钱 / 伪清醒模板**
- **开头三件套：** 痛点 + 身份标签 + 时间锚点
  - `25 岁裸辞第 30 天，我终于明白...`
  - `建议所有内核不稳的女生反复观看`
  - `30 岁之后我才懂，真正拉开差距的是...`
- **灵性 / 心理学黑话稀释版：**
  - 允许一切发生、把自己重养一遍、能量场、课题分离、内核稳定、高能量人格
  - 戒掉讨好型人格、MBTI 平替（`i 人 / e 人 / 淡人 / 浓人 / 确诊为 xx`）
- **搞钱黑话：** 复利思维、认知杠杆、长期主义、睡后收入、反内卷
- **排版特征：** 每段开头一个 emoji，三段式（痛点 → 方法 → 升华），结尾必带 `建议收藏 / 慢慢看 / 评论区告诉我你的阶段`
- **诊断关键：** **正确的废话 + 排比句 + 情绪平滑曲线**,但没有具体事实、约束或行动。原文没有细节时只报告缺口,不得为了“更像人”补写账单或经历。

**C. 朋友圈装 X 文案（凡尔赛式）：**
- `今天的阳光真好☀️,随手拍了一张📷,没想到竟然火了🔥`
- 刻意"无心" + 滤镜 + 留白
- `今天开了个会，老板把我从 xx 提到 xx，心情复杂` 这种

**D. 短视频口播（2025 算法驯化版）：**

**老套路（基本死透）：**
- `今天给大家分享一个超级实用的小技巧`
- `不看后悔!看完赶紧转发给身边的朋友`

**新 AI 腔：算法钩子体**
- **玄学 / 大数据钩子：**
  - `刷到这条视频，说明你马上要...`
  - `大数据不会乱推，留下"接"字...`
  - `能刷到这条的人都不简单`
- **恐吓 / 反常识钩子：**
  - `千万别再 X 了! 90% 的人都在犯这个错`
  - `我劝你立刻停止 X`
  - `这三件事，你现在知道还不晚`
- **电影解说腔（AI 重灾区，永不停歇）：**
  - `注意看，这个男人叫小帅 / 小美...`
  - `这个看似不起眼的动作，背后藏着...`
- **节奏特征（AI 配音可辨）：**
  - 句长被严格控制在 **3-5 秒一个断句**（方便 AI 配音换气）
  - 每 15 秒必有一个**情绪转折点**
  - 结尾 CTA 机械化：`点赞收藏，不然刷着刷着就找不到了 / 关注我，带你了解更多`

**E. AI 客服"您好呢亲":**
- `您好呢亲~很高兴为您服务哦!`
- `亲亲这边建议您...`
- `亲,这个问题小助手已经记下啦~`

**改写方向：**
- **伪深度咨询腔** → 抽掉术语后能不能保留信息？不能就整段重写，换成具体公司 / 数字 / 事件
- **伪疗愈 / 伪搞钱** → 原文有具体细节就把抽象升华落回细节;原文没有就删空话并报告缺口,不补写经历
- **算法钩子 / 电影解说腔** → 删钩子，直接给内容；本来就没内容，建议别发
- 删密集 emoji、机械 CTA、刻意惊呼
- 回归真实语气；原本就不真实，建议不改写，而是建议用户别发
- **万能赞美词降密度** → 短段 <300 字里 `宝藏 / 神器 / 天花板 / 绝了 / 绝绝 / 绝绝子 / 好吃到想原地开店 / 气场两米八` 累计 ≥3 处时,只保留**紧扣具体事实**的那处(配合后面跟具体菜名 / 成分 / 价格的),其余改为具体描述或删掉
- **身份标签 + 反向 CTA 降密度** → `姐妹们快冲 / 不好吃回来打我 / 打工人必看 / 租房党必看 / 闭眼入` 组合 ≥2 处时,全篇最多保留一处;开头 + 结尾双号召的刚性模板必须拆掉一处
- **info-block emoji 微模板** → 保留信息槽位(`必点 / 地址 / 价格 / 晨晚护肤` 是合理的小红书阅读线索),但**删 emoji 前缀 + 把四字小标题压成自然短语**:`✅ 必点招牌` → `我点的几样`、`🌟 晨间护肤 / 🌙 晚间护肤` → `早上怎么洗 / 晚上怎么洗`、`✅ 技巧 1 / ✅ 技巧 2` → `技巧 1 / 2`(去掉 ✅)

**关于小红书 / 微博 hashtag 尾巴:**

hashtag 只按相关性、目标平台和用户要求处理。平台名冗余标签与大量泛标签会稀释主题,但数量本身不用于判断来源。改写规则:

- **删:** 与内容无关或只重复平台名的 `#小红书 X / #X 薯 / #爆款 / #日常 / #分享`
- **保留:** 话题化标签(`#上海美食 / #穿搭 / #护肤 / #租房改造`)
- **控制数量:** 按内容主次排序,只留目标平台需要且最相关的标签;用户有投放规范时服从规范

**例:** 原文 `#上海美食 #魔都探店 #面馆推荐 #平价美食 #吃货薯 #周末去哪儿 #我的美食日记 #小红书爆款美食`(8 个 + 2 个平台冗余) → 改后 `#上海美食 #探店 #面馆`(3 个,话题化)

> **语体限定：** 这条**只在自媒体 / 社交语体里激活**。商务、学术、公文、**品牌广告**里不会出现这些，不需要扫。**尤其注意:品牌广告 / 文案语体（Apple / Nike / 李宁 官网与 campaign）跟自媒体营销号不是同一个物种,不要把"#37 自媒体套路"往品牌文案上套**;详见"第零步 · 品牌广告 vs 自媒体营销号：三条识别线"。

> **关于"底层逻辑 / 颗粒度 / 心智"等词：** 这些词在**真·产品经理写 PRD、运营写复盘**时就是工作语言，不是 AI 腔。只有在**自媒体语境**里、且伴随"**伪深度结构**"（抽掉术语丢信息）时，才触发 #37-A。工作场景里不扫。

### 38. 文学 / 故事类 AI 味

**A. 故事模板开头：**
- `从前有座山,山里有座庙,庙里有个老和尚...`
- `在很久很久以前的一个小村庄里...`
- `那是一个风雨交加的夜晚...`
- `他永远忘不了那个下午...`

**B. 空意象 / 对仗型文字：**
- 对仗工整但无感情：`月光如水水如天,思绪如云云如烟`
- 堆叠万能意象：`风、月、雨、雪、山、水、云、影`
- 主题空泛：`乡愁 / 岁月 / 时光 / 人生`

**问题：** 模板化开头、空意象和工整对仗如果不能承载具体观察,会让“文学感”停在形式上。

**改写方向：**
- 换掉模板开头，用具体时间、地点、人起句(`1978 年冬天,我爸在工厂门口抽烟`)
- 意象换成有指向性的观察（不是"月光",是"透过旧窗帘的那点光"）
- 允许不对仗、不整齐

> **语体限定：** 这条**只在文学 / 虚构 / 诗意类文本里激活**。非文学内容（论文、报告、邮件、朋友圈）不扫。

### 49. 网络流行语机械套用：毒性正能量缝合

> **核心辨识点：** 比起单纯的密度、错位或过时,更值得处理的是负面流行语突然接正能量结论,造成语气撕裂。

**核心问题：毒性正能量缝合**

「打工人 / 内卷 / 躺平 / 班味」通常承载自嘲、反讽或疲惫。后面如果机械接励志、自我和解或长期主义,词的情绪会和句子落点对不上。

**缝合句式(高辨识度 AI 腔):**

- `作为一个打工人/牛马,我想说:` + 积极行动建议
- `虽然班味很重,但只要 X,就能找回 Y` 型的励志转折
- `内卷是没有出路的,真正的答案是长期主义/自我价值/向内生长`
- `躺平不是终点,而是重新出发的起点`
- `吗喽的命也是命,所以我们更要 X`

问题不在于能不能升华,而在于转折是否有原文事实支撑。没有支撑的“累完必升华”按空转处理。

**三种诊断信号:**

1. **信号一 · 毒性正能量缝合** — 流行语 + 180° 转折 + 积极结论。见上方句式。**单次出现即警觉**,因为这是句式级的 AI 腔,不是密度级的。
2. **信号二 · 密度超标** — 一段 <200 字里流行语出现 ≥3 次,且集中在开头或每段首句。
3. **信号三 · 语境错位** — 商业计划书、政策段、学术文、PRD、述职报告里出现 `yyds / 懂得都懂 / 吗喽`,且与受众和声口不匹配。

**关键词时效参考(使用时需按目标社群和当前语境复核)**

| 分类 | 举例 | 现状 |
|---|---|---|
| **日常基础词** | 破防、蚌埠住了(绷不住)、emo、打工人、内卷、躺平 | 不凭单词触发;看缝合、密度与语体 |
| **阶段性流行词** | 班味、松弛感、数字游民、活人感、预制 xx | 时效会变化,使用前按目标社群复核 |
| **社群口语** | 抽象、牛马、吗喽、淡人 / 浓人、确诊为 xx、包 xx 的 | 社群差异大,只看是否符合受众与声口 |
| **高复用旧模板** | yyds、绝绝子、栓 Q、凡尔赛文学、家人们谁懂啊、姐妹们快冲、不好吃回来打我 | 不能仅凭出现触发;与 emoji、身份号召和空转结构成簇时再处理 |
| **梗 / 缩写** | 小丑竟是我自己、活久见、xswl、gkd、nbcs | 自然玩梗可保留,密集堆叠才处理 |

**改写方向：**

- **毒性缝合** → 把升华那半句砍掉,留下流行语本身的情绪色调。累就累,不用接"但我还是相信自己值得更好"。
- **密度高** → 全段只留最合理的一处,其它改具体表达。"yyds" → 说清楚好在哪。
- **语境错位** → 正式文体里直接删,用符合语体的中性说法。
- **已入土词** → 换成 2025 鲜活词,或直接去掉,用朴素说法。

**原文(毒性正能量缝合 · 最典型的新 AI 腔):**
> 作为一个每天通勤两小时的打工人,我也曾班味浓得化不开。但后来我明白了:内卷是没有出路的,真正的答案在向内生长。与其焦虑,不如允许一切发生,慢慢把自己重养一遍。

**改后:**
> 我每天通勤两小时,回家后什么也不想做。原文没有给出解决办法,这里不补写「向内生长」之类的答案。

**原文(密度过高):**
> 这个功能真的 yyds！用完懂得都懂,姐妹们冲就完了！真的绝绝子,不冲我真的会谢。

**改后:**
> 这个功能挺顶的 —— 之前要点三层菜单,现在一键到底。

**原文(语境错位):**
> 本季度公司业绩实现重大突破,员工们表示 yyds,后续将继续保持这种状态。

**改后:**
> 本季度营收同比涨 23%,主要来自海外。

> **例外(不触发):**
> - 一段文字(>200 字)里自然出现一次流行语,且语义准确
> - 聊天 / 对话 / 朋友圈这种本就偏口语的文体,密度容忍度高
> - 自嘲型、反讽型使用且不接空转升华,如「今天班味真重,下班」
> - 判断标准:流行语是否准确承载当前语气,转折后有没有具体依据。没有依据才处理

### 50. B 站 AI 味(视频简介 / 科普解说 / 机械玩梗 / AI 字幕)

B 站的视频简介、科普解说、玩梗和字幕各有特定语体。本节处理模板化与事实风险,不把平台痕迹当作者身份判断。

**A. 视频简介 AI 模板(含时间轴幻觉)**

- 套路固定:**一句开场钩子 → 三行内容概要 → 时间轴 → 结尾求三连**
- **时间轴事实风险** —— 章节时间(`03:27 重点讨论 XX`)必须与视频逐项核对;没有视频证据时不得补写
- 结尾模板:`感谢大家的观看,如果这期视频对你有帮助,别忘了一键三连支持一下UP主哦!`

**B. 科普 / 解说稿(比喻 + 升华的 AI 体操)**

- **比喻成瘾:** "你可以把 X 理解成 Y" / "这就像..." —— 每段必来一个,而且比喻经常不贴切
- **强行上价值:** 讲一个具体技术 / 事件 / 人物,结尾必升华到"人类文明 / 科技浪潮 / 时代命题"
- **并列三件套结构:** 「原因主要有三个:第一...,第二...,第三...」只有在内容确实分三项时才保留
- **"接下来让我们一起来看看 / 下面我们就来聊一聊"** —— 口播过渡句的 AI 味重灾区

**C. 机械玩梗 / 电波错位**

- AI 硬套 B 站梗,但用错时代 / 用错语境:`AWSL`、`3Q`、`这集我看过`、`素质三连`、`爷青回` 用在不匹配的内容上
- **梗已入土还在用:** `真香` / `奥利给` / `哈基米` 套在 2025 的内容里,像两年前的复读
- **弹幕化台词混进正文:** 书面内容里连续出现 `人类还在用爱发电吗` / `下次一定`,且与正文声口不一致

**D. AI 翻译字幕残留**

- `我们...`、`我们的...` 主语代词密度异常(英文 `we/our` 直译,见模式 #33)
- 长定语句直译成一串"的": `一个被认为是...的...的...的 X`
- 半角标点混入全角环境(`,` 和 `，` 混用)
- 英文专有名词没译或乱译:`Transformer 模型(变压器模型)`、`context window(语境窗口)` 强行中译

**问题:** 口播、简介、字幕和弹幕是不同载体。把其中一种载体的套话直接搬进另一种,会造成声口错位;时间轴和技术名还涉及事实核验。

**改写方向：**

- **简介** → 删套模板,直接一两句说清楚这期讲什么;时间轴要么不写,要么对着视频真核对后再写(幻觉时间轴比没有时间轴糟)
- **口播稿** → 保留 UP 主的个人口头禅、停顿、跑题;把"强行升华"那段砍掉;比喻只留最贴切的一个
- **梗** → 只留跟具体内容真相关的一个;跨代际的梗先查近半年的 B 站热度再用
- **翻译字幕** → 按 #33 翻译腔残留办
